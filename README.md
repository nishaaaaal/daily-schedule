
Daily Schedule

A simple interactive daily schedule that shows timers, progress bars, and highlights the current task based on your system time.

Features
	•	Real-time countdown timers
	•	Morning / Afternoon / Evening sessions
	•	Automatically marks tasks as Past, Active, or Upcoming
	•	Expand / Collapse task details
	•	“Jump to Now” button
	•	Works offline (just open the HTML file)

How to Use
	1.	Open the file gemi.html in any browser
	2.	Or rename it to index.html and host it with GitHub Pages
	3.	The page will automatically track time and update your schedule
